## 🚀 Getting Started

### View Locally

```bash
# Clone the repository
git clone https://github.com/mukilanchitrakumar/Portfolio.git

# Open in browser
cd Portfolio
open index.html
```

> No build step, no npm install, no server required. Just open `index.html` in any modern browser.

### Deploy Online (Free)

| Platform | Steps |
|---|---|
| **Vercel** | Drag & drop the folder at [vercel.com/new](https://vercel.com/new) |
| **Netlify** | Drag & drop at [app.netlify.com/drop](https://app.netlify.com/drop) |
| **GitHub Pages** | Go to repo Settings → Pages → Deploy from `main` branch |
